Help me organize recent screenshots on my Desktop.

Phase 1: Initial Desktop Scan

First, scan my Desktop and count how many screenshots/images are there. Show me:

  • Total count
  • Date range (oldest to newest)

Phase 2: Analyze Recent Screenshots

Then, focus only on screenshots from the last 14 days. For each one:

  • Identify what it shows
  • Suggest a descriptive filename
  • Propose which folder it belongs in (or if it can be deleted)

Phase 3: Organization Plan

Group similar screenshots together. Show me the plan before making any changes.

Phase 4: Preview Organization

After I approve, start by organizing just 10 files as a preview. If there are more than 10 files, check in with me before continuing with the rest.

Key Constraints

  1. Only process files from the last 14 days in Phase 2 onwards
  2. Show the complete plan for my approval before making any file changes
  3. Organize exactly 10 files in the preview phase (or fewer if total count is less than 10)
  4. Wait for explicit confirmation before proceeding with remaining files

Expected Output Format

For each screenshot, provide:

  • Current Name: Original filename
  • Description: What the screenshot contains
  • Suggested Name: New descriptive filename
  • Destination: Target folder or deletion recommendation
