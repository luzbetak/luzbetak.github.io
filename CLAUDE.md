# CLAUDE.md

## Project Overview

Personal technical knowledge base and portfolio website for Kevin Thomas Luzbetak, MSc (Data & Machine Learning Engineer). Static GitHub Pages site at `https://luzbetak.github.io`.

## Tech Stack

- **Static Site Generator:** Jekyll (GitHub Pages compatible)
- **Frontend:** HTML, CSS, JavaScript
- **Build Scripts:** Python 3.11+ (menu generation, search indexing)
- **Code Highlighting:** Prism.js (Python, YAML, SQL, Markdown, Bash, Go)
- **Client-Side Search:** Lunr.js with NLP-generated index (spaCy, NLTK)
- **Theme:** Dark (black `#000`, white text, blue nav `#0066cc`)

## Page Template Convention

Every content page follows this pattern:

```html
---
---
{% include menu.html title="Page Title" %}

<!-- Content goes here -->

{% include footer.html %}
```

- The empty front matter (`---\n---`) is required for Jekyll processing.
- The `title` parameter in menu.html sets the page title and breadcrumb.

## Build Commands

```bash
# Regenerate navigation menu from directory structure
python 1-menu-generator.py

# Regenerate search index (requires spaCy, NLTK, BeautifulSoup)
./2-index-website.sh

# Deploy (GitHub Pages auto-deploys from main branch)
gh run watch --exit-status
```

## Key Conventions

- **Navigation menu** is auto-generated by `1-menu-generator.py` — do not edit `_includes/menu.html` directly.
- **Search index** is auto-generated by `page_index.py` — do not edit `search/search-index.json` directly.
- Content directories each have an `index.html` as their entry point.
- Code blocks use Prism.js classes: `<code class="language-python">`, `language-sql`, etc.
- CSS uses fixed width `1100px`, dark backgrounds (`#000`, `#222`, `#333`), and hover color `#ffcc00`.
- After adding/removing/renaming pages, re-run `1-menu-generator.py` then `2-index-website.sh`.
