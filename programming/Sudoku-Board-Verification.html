Sudoku Board Verification

#!/usr/bin/env python
import pprint

# ----------------------------------------------------------------------------#
# Define a 9x9 matrix representing a Sudoku board
matrix1 = [
    [7, 1, 8, 5, 3, 2, 9, 4, 6],
    [5, 3, 2, 6, 9, 4, 1, 8, 7],
    [6, 9, 4, 7, 1, 8, 3, 2, 5],
    [1, 2, 7, 3, 4, 5, 8, 6, 9],
    [9, 8, 6, 1, 2, 7, 4, 5, 3],
    [3, 4, 5, 9, 8, 6, 2, 7, 1],
    [4, 6, 3, 8, 7, 9, 5, 1, 2],
    [8, 7, 9, 2, 5, 1, 6, 3, 4],
    [2, 5, 1, 4, 6, 3, 7, 9, 8],
]

# ----------------------------------------------------------------------------#
def verify_sudoku_board(board, value):
    """
    Verifies whether a given 9x9 Sudoku board is valid by checking that
    each row, column, and 3x3 sub-grid (window) sums to a specified value.

    Args:
        board (list of list of int): The 9x9 Sudoku board represented as a list of lists.
        value (int): The expected sum of each row, column, and 3x3 sub-grid.

    Returns:
        bool: True if the board is valid according to Sudoku rules, False otherwise.
    """
    
    # Verify each row sums to the given value
    for row in range(0, 9):
        if sum(board[row]) != value:
            return False

    # Verify each column sums to the given value
    for col in range(0, 9):
        total = 0
        for row in range(0, 9):
            total += board[row][col]
            # After finishing a column, check if the total sum matches the expected value
            if row == 8:
                if total != value:
                    return False

    # Verify each 3x3 sub-grid (window) sums to the given value
    for start in range(0, 9, 3):
        total = 0
        for col in range(start, start + 3):
            for row in range(start, start + 3):
                total += board[row][col]

        if total != value:
            return False

    return True

# ----------------------------------------------------------------------------#
if __name__ == "__main__":
    pp = pprint.PrettyPrinter()
    pp.pprint(matrix1)
    
    result = verify_sudoku_board(matrix1, 45)
    print(result)

Explanation

Matrix Definition (matrix1):

This is a 9x9 matrix representing a completed Sudoku board. Each inner list represents a row in the Sudoku puzzle.

Function verify_sudoku_board(board, value):

Purpose: This function checks if a given 9x9 Sudoku board is valid. It verifies that:

  • The sum of numbers in each row equals the expected value.
  • The sum of numbers in each column equals the expected value.
  • The sum of numbers in each 3x3 sub-grid (also known as a "window") equals the expected value.

Parameters:

  • board: The 9x9 list of lists representing the Sudoku board.
  • value: The expected sum for each row, column, and 3x3 sub-grid. For a standard Sudoku puzzle, where the numbers 1 through 9 are used, this value should be 45 (since the sum of 1 to 9 is 45).

Returns:

True if the board is valid according to the above conditions, otherwise False.

Logic:

  • Rows: The function sums the numbers in each row and checks if it matches the expected value.
  • Columns: The function sums the numbers in each column and checks if it matches the expected value.
  • 3x3 Sub-Grids: The function sums the numbers in each 3x3 sub-grid and checks if it matches the expected value.
