Patterns Analysis Prompt

Help me find patterns and insights across my files.

I have messy, unstructured files and want to understand what themes are emerging.

Phase 1: Folder Scan and Summary

First, scan the folder and show me a summary:

  • Total files
  • Date range (oldest to newest)
  • Types of content

Phase 2: Clarifying Questions

Before analyzing, ask me:

  • What I'm hoping to discover (recurring themes, contradictions, evolution of thinking, action items, or something else)
  • Whether certain files or time periods should be prioritized
  • What format would be most useful for the final analysis

Phase 3: Initial Pattern Analysis

If there are more than 20 files, start by analyzing just the 10 most recent files.

Show me the top 3–5 patterns you found with 2–3 specific examples for each pattern.

Pattern Presentation Format

For each pattern, include:

  1. Pattern name: A clear label for the theme
  2. Description: What this pattern represents
  3. Examples: 2–3 specific quotes or references from the files
  4. Frequency: How often this pattern appears in the analyzed files
  5. Significance: Why this pattern matters

Phase 4: Validation and Continuation

Once I confirm you're on the right track, analyze the remaining files.

Key Process Notes

  • Do not analyze all files at once if there are more than 20—start with the 10 most recent
  • Wait for confirmation before proceeding to remaining files
  • Ground all patterns in specific examples from the source material
  • Prioritize files and time periods as indicated by the user
  • Format the final analysis according to user preferences

Supported File Types

Analyze content from:

  • Voice memos (transcribed or audio)
  • Meeting transcripts
  • Documents and notes
  • Journal entries
  • Any other folder specified by the user
